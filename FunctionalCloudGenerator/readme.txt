Command Line Arguments format:
[text file] [banned words file] [configuration file] [result file name (without extension)]

Configuration file format:
[width] [height]
[text font]
[minimal font size] [maximal font size]
[background color]
[text color 1] [text color 2] ... [text color N]
[result file extension]
[algorithm (simple or random)]
[text file format]
[maximal amount of words]
[application type (console)]
